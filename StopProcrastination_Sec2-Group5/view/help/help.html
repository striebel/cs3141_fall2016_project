Stop Procrastination User Documentation

Don't succumb to online distractions; we allow you to block sites that cause procrastination.

No-Block

No-Block is Stop Procrastination's most lenient block-type. When No-Block is active, no website will be blocked. No-Block is activated and deactivated from Stop Procrastination's main popup menu.

Blacklist

Blacklist is a block-type that allows you to create a list of procrastination causing sites, which will always be blocked, while all sites not on the list are allowed. Blacklisting is activated and deactivated from Stop Procrastination's main popup menu. Your blacklist can be edited under the settings menu's "Blacklist" tab.

Whitelist

Whitelist is a more aggressive block-type that allows you to create a list of allowed sites, which are the only sites that will not be blocked. Whitelisting is activated and deactivated from Stop Procrastination's main popup menu. Your whitelist can be edited under the settings menu's "Whitelist" tab.

Blackout

Blackout is Stop Procrastination's most strict block-type. When blackout is active, all websites are blocked. Blackout is activated and deactivated from Stop Procrastination's main popup menu.

Basic Timer

The Basic Timer allows you to specify an amount of time for which you would like to current block-type to persist. Once the Basic Timer elapses, the block-type will be set to No-Block. The timer may be set, viewed, and canceled from the main popup menu. The Basic Timer cannot be set when a Daily Timer interval is active.

Daily Timer

Stop Procrastination's Daily Timer adds a level of sophistication to the Basic Timer by allowing recurring time intervals to be set when blacklisting or whitelisting will be active. For example, the Daily Timer could be set so that blacklisting will be active from 3-6pm on Mondays. The Daily Timer supports any number and combination of intervals, so long as no two intervals overlap. The Daily Timer can be edited under the settings menu's "Daily Timer" tab.
